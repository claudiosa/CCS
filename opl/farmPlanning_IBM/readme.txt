Farm Planning

Problem 8 from Model Building in Mathematical Programming by
H.P. Williams. Calculate how the farmer should operate over the next
five years to maximize profit.


How to run this example?
  * In CPLEX Studio IDE, import the example "Farm planning" 
right click on the run configuration "Basic Configuration" and select "Run this"
  * In the command line, execute "oplrun -p <Install_dir>\opl\examples\opl\models\FarmPlanning"
