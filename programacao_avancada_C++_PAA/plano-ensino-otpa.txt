Ementa
Fundamentos matemáticos. Programação e linguagens. Estruturas de dados. 
Sistemas de Arquivos. Algoritmos de buscas em Strings. Recursividade. 
Ordenação. Busca exaustiva. Método guloso. Divisão-e-conquista. 
Programação dinâmica. Grafos. Programação com Backtracking. 
Implementação de soluções diversas para ampla variedade de problemas. 

Objetivo Geral

Prover ao estudante um arcabouço de resolução de problemas computacionais do tipo desafios de
programação.

Objetivo Especificos
- Aprender os fundamentos de uma linguagem de programação utilizada nas competições da
  Maratona de Programação
- Retomar os fundamentos teóricos sobre os conteúdos da ementa proposta
- Aprender a realizar implementações destes problemas na linguagem apresentada
